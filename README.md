# VueZ

Affichage des altitudes (Z) de tous les sommets des entités d’un layer (ponctuel, ligne, polyligne, polygone).

## Pré-requis

Mandatory : The "plugin\_maitre" must be installed.
Link to the plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)

## Fonctionnalités

French : 
Affichage des altitudes (Z) de tous les sommets des entités d’un layer.
Le layer peut être de n’importe quel type (ponctuel, ligne, polyligne, polygone).

La visualisation se base sur le layer actif et est limitée à la zone visible à l’écran.
Il suffit de relancer le plugin si la zone visible a changé.

English :
Display of altitudes (Z) of all vertices of the features in a layer.
The layer can be of any type (point, line, polyline, polygon).

The visualization is based on the active layer and is limited to the visible area on screen.
Simply relaunch the plugin if the visible area has changed.
